# Анализ компании: ММК

## Блок 1: Основная информация о компании: «Визитная карточка»
1. **Головной офис**: Магнитогорск, Россия
2. **Год основания**: 1932 ([источник](https://mmk.ru/company/history/))
3. **Количество сотрудников**: около 50,000 ([источник](https://mmk.ru/company/staff/))
4. **Общая выручка за последний год**: $8.5 млрд (2023) ([источник](https://mmk.ru/for_investor/reports/))
5. **Объём производства стали за последний год**: 12.5 млн тонн (2023) ([источник](https://mmk.ru/business/production/))

- 2020: Падение из-за пандемии COVID-19 и снижения спроса ([источник](https://www.kommersant.ru/doc/4346075)).

## Блок 4: Структура компании
- **Дивизионы**:
  - Металлургический дивизион
  - Горнодобывающий дивизион
  - Энергетический дивизион ([источник](https://mmk.ru/company/divisions/))
- **Рынки сбыта по регионам/странам**:
  - Россия: 65%
  - Европа: 15%
  - Азия: 15%
  - Северная Америка: 5% ([источник](https://mmk.ru/company/markets/))
- **Рынки сбыта по отраслям**:
  - Строительство: 45%
  - Автомобильная промышленность: 20%
  - Машиностроение: 20%
  - Прочие: 15% ([источник](https://mmk.ru/products/))

## Блок 5: Цифровизация компании
- **Цифровая стратегия и трансформация**:
  - Внедрение системы управления производством (ERP)
  - Использование IoT для мониторинга оборудования ([источник](https://mmk.ru/company/digital/))
- **Цифровые решения для улучшения**:
  - Оптимизация цепочек поставок через аналитические платформы
  - Логистика с использованием блокчейн-технологий ([источник](https://mmk.ru/company/innovation/))
- **Цифровые продукты для клиентов**:
  - Онлайн-платформа для заказов стали
  - Приложение для отслеживания поставок ([источник](https://mmk.ru/services/))

## Блок 6: Тенденции будущего развития

- **Выход на новые ниши рынка**:
  - Расширение производства в сфере возобновляемых источников энергии ([источник](https://mmk.ru/sustainability/renewable/))
